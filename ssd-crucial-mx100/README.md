# Crucial MX100 SSD (128GB)

## Device info

 * [`Marvell 88SS9189` SoC](https://www.marvell.com/storage/assets/Marvell_88SS9187-001.pdf).
 * 8 `Micron NW592` NAND chips.
 * `MSP430` and one serial (SPI) flash labeled `25P16`.
